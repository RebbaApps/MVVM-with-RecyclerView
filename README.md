# MVVM-with-RecyclerView
“A simple Android app built using MVVM architecture with Retrofit for API integration. The project displays user data in a RecyclerView using CardViews and ConstraintLayout. It demonstrates clean code structure, ViewModel–LiveData usage, and efficient data binding for smooth and maintainable UI updates.”
